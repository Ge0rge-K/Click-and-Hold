# Click-and-Hold

Simple Toggle Left click mouse button down. 

To install navigate to the Releases Catagory and download Click_Gui.EXE
Once Downloaded run the file and done!


If you want to build yourself follow these directions...

Req: Have python 3.14+ installed
Req: If you are self building install libraries

1) Download the file
2) Once downloaded extract the file
3) One you are in the your folder right click and open terminal
4) Once in terminal run python Click_and_Hold.py
5) Done!
